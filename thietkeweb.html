sử dụng công cụ này để thực hiện các thay đổi ở cấp độ nâng cao trên toàn hệ thống. Bạn có thể nhập các lệnh
dựa trên văn bản để tự động hóa một loạt tác vụ.
Dưới đây là các bước chi tiết để sử dụng Command Prompt tạo nhiều thư mục cùng một lúc. Đảm bảo bạn đã đăng nhập vào Windows
với quyền admin trước khi tiếp tục:
1. Nhập Command Prompt trong tìm kiếm Windows và nhấp vào Run as administrator.
2. Ngoài ra, bạn cũng có thể mở Runn bằng cách nhấn Win + R và nhập cmd vào trường văn bản.
